Technical Improvements Breakdown

  • Typography System: Integrated Google Fonts ('Outfit') for a modern, executive look. Controlled font-weight and line-height for readability.
  • Layout Architecture: Replaced linear vertical stacking with a 3-column symmetric Grid for Skills and a Flexbox-based sticky Navigation Bar.
  • Theming & Palette: Implemented "Obsidian Noir" (Dark Theme) using consistent HSL color tokens and gold highlights (#FFD700).

    Technical Deep-Dive: Styling Improvements

    Implementation of Symmetric Skills Grid

    Transitioned from simple <ul> bullets to a complex interactive grid layout.

    /* Symmetric 3-Column Layout */ .skills-grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 40px; margin: 50px 0; } .skill-card { background: rgba(255, 255, 255, 0.05); border-radius: 12px; padding: 30px; transition: transform 0.3s ease; }

    Premium Responsive Navigation

    Added sticky positioning with blur-effect backdrop for a high-end app feel.

    nav { position: sticky; top: 0; z-index: 1000; background: rgba(10, 10, 10, 0.85); backdrop-filter: blur(12px); display: flex; justify-content: space-between; align-items: center; }

    Experience Timeline & Positioning

    Developed a custom vertical timeline using pseudo-elements and relative positioning.

    .timeline-item { position: relative; padding-left: 35px; border-left: 2px solid #FFD700; margin-bottom: 50px; transition: opacity 0.5s ease; } .timeline-item::before { content: ''; position: absolute; left: -7px; top: 0; width: 12px; height: 12px; background: #FFD700; }

    "By utilizing CSS Selectors (Class, ID, and Element), I moved from a tagged-text document to a product-architected interface. The use of variables and selectors ensures that the site is not only beautiful but also maintainable and scalable."
